About Us

This is a student mini project focused on building a housing price prediction website using Machine Learning and Flask. The goal is to create a clean, responsive design that allows users to explore properties and predict house prices easily.

Project Purpose

To build a real-world ML-based web application using Flask and deploy a predictive model.

Design Focus

Modern layout with smooth navigation, attractive visuals, and mobile-friendly components.

Student Contribution

Complete backend, frontend, and ML model development done as part of a learning journey.
